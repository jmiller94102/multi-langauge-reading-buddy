## Component Breakdown

### **Item Card (Food)**
- Image/Icon (emoji or image)
- Name
- Rarity badge (Common/Uncommon/Rare/Epic)
- Effects list
- Price (coins or gems)
- Purchase button
- Hover: Show detailed effects

### **Item Card (Cosmetic)**
- Image/Icon
- Name
- Evolution track requirement
- Unlock status (locked if level requirement not met)
- Price
- Preview button (shows pet with cosmetic)

### **Item Card (Power-Up)**
- Icon
- Name
- Description
- Duration/uses
- Current owned count
- Price
- Purchase button

### **Treasure Chest Card**
- Icon
- Name
- Contents list with ranges
- Price
- Daily/weekly limit
- Purchase button

### **Category Tabs**
- Icon + label
- Active state indicator
- Scrollable on mobile

### **Balance Display**
- Prominent coin and gem counts
- Updates in real-time after purchase

### **Pet Favorites Section**
- Shows pet's preferred food type
- Recently fed history
- Food collection progress
- Encourages trying new foods

## Accessibility

- **Keyboard Navigation**: Tab through items, Enter to purchase
- **Screen Reader**: Announce item name, effects, price, owned count
- **High Contrast**: Rarity colors distinguishable
- **Focus Indicators**: Blue outline on focused item
- **Confirmation Modals**: Prevent accidental purchases

---

## Performance

- **Lazy Load Images**: Load item images as user scrolls
- **Cache Prices**: Don't re-fetch item catalog on every visit
- **Debounce Purchases**: Prevent double-purchases
- **Optimize Icons**: Use sprite sheets for emojis

---

## Notes

- Shop should feel **rewarding but not manipulative**
- Cultural foods **teach appreciation** for different cuisines
- Power-ups are **optional, not required** for progression
- Treasure chests add **excitement and surprise**
- Limited specials create **urgency** but with fair pricing
- Balance display is **always visible** to prevent overspending
- Cosmetics are **purely aesthetic** (no pay-to-win)
